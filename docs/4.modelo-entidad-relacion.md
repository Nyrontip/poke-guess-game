# 📊 Modelo Entidad-Relación

El diagrama entidad-relación se visualiza sirviendo la carpeta `docs/erd/` con cualquier servidor estático:

```bash
npx serve docs/erd/
```

Esto abre un visualizador interactivo (Liam ERD) con el schema completo definido en `docs/erd/schema.json`.

### Alternativas

```bash
# Con live-server
npx serve docs/erd/

# Con Python
python3 -m http.server 3000 -d docs/erd/
```

### Schema

El modelo tiene 2 tablas:

- **Player** — jugadores registrados (`id`, `name`, `avatar`, `created_at`)
- **Score** — resultados de partidas (`id`, `player_id` → Player, `correct`, `total_time_ms`, `score`, `date`, `created_at`)

Para ver tipos exactos, constraints, índices y relaciones, abrí el `schema.json` en `docs/erd/schema.json`.
